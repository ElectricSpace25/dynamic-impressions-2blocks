# Dynamic Impressions jsPsych 2 Blocks

This is ajsPsych study based on the [Dynamic Impressions Template](https://github.com/ElectricSpace25/dynamic-impressions-template). It features two blocks, each consisting of their own instructions and video trials, that are presented in a random order.

Visit [the template wiki](https://github.com/ElectricSpace25/dynamic-impressions-template/wiki) for documentation.
